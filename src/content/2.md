# Лекция 2: Модели, описываемые одним дифференциальным уравнением первого порядка

## 1. Основные понятия
*   **Общий вид уравнения:** \\(\frac{dx}{dt} = f(x, t)\\).
*   **Автономное уравнение:** Уравнение вида \\(\frac{dx}{dt} = f(x)\\), где правая часть не зависит явно от времени. Состояние такой системы в любой момент характеризуется одним значением переменной \\(x\\).
*   **Интегральная кривая:** Графическое представление решения \\(x = x(t)\\) на плоскости \\((t, x)\\). Согласно теореме Коши, интегральные кривые одного уравнения не могут пересекаться.

## 2. Фазовое пространство
*   **Фазовые переменные:** Значения переменных системы, отложенные на координатных осях.
*   **Фазовое пространство:** Пространство, в котором отображаются состояния системы. Для одного уравнения это **фазовая прямая**.
*   **Фазовая траектория:** Линия, вдоль которой движется система в фазовом пространстве.

## 3. Стационарное состояние (состояние равновесия)
*   Это состояние, в котором значения переменных не меняются со временем.
*   **Условие:** Скорость изменения равна нулю: \\(\frac{dx}{dt} = 0\\), следовательно, \\(f(x) = 0\\).
*   На графике \\((t, x)\\) стационарные состояния выглядят как горизонтальные прямые (асимптоты).

## 4. Устойчивость по Ляпунову
*   **Устойчивое состояние:** Если при малом отклонении от него система не уходит далеко от особой точки.
*   **Асимптотически устойчивое состояние:** Если малые отклонения со временем затухают (система возвращается в равновесие).
*   **Неустойчивое состояние:** Если малые отклонения со временем увеличиваются.
*   **Аттрактор:** Множество, к которому стремятся траектории системы (притягивающее множество). Для одного уравнения простейший аттрактор — устойчивая точка покоя.

## 5. Методы исследования устойчивости

### А. Аналитический метод (Метод линеаризации Ляпунова)
1.  Рассматривается отклонение \\(\xi\\) от стационарного состояния \\(\bar{x}\\).
2.  Уравнение линеаризуется с помощью разложения в ряд Тейлора.
3.  Получается уравнение первого приближения: \\(\frac{d\xi}{dt} = \lambda\xi\\), где **\\(\lambda = f'(\bar{x})\\)** (производная правой части в стационарной точке).
    *   Если **\\(\lambda < 0\\)** — состояние **устойчиво**.
    *   Если **\\(\lambda > 0\\)** — состояние **неустойчиво**.
    *   Если **\\(\lambda = 0\\)** — метод первого приближения не дает ответа.

### Б. Графический метод
Устойчивость определяется по графику функции \\(f(x)\\):
*   Если \\(f(x)\\) меняет знак с **«+» на «–»** при возрастании \\(x\\) — точка **устойчива**.
*   Если \\(f(x)\\) меняет знак с **«–» на «+»** — точка **неустойчива**.

## 6. Примеры моделей

### Рост колонии микроорганизмов (Экспоненциальный рост)
*   **Уравнение:** \\(\frac{dx}{dt} = rx\\) (где \\(r\\) — константа скорости роста).
*   **Решение:** \\(x = x_0 e^{rt}\\).
*   При \\(r > 0\\) — неограниченное размножение, при \\(r < 0\\) — вымирание.

### Вещество переходит в раствор (Насыщение)
*   **Уравнение:** \\(\frac{dx}{dt} = k(P - x)\\).
*   **Решение:** \\(x = P - C_1 e^{-kt}\\).
*   График представляет собой кривую с насыщением, стремящуюся к пределу \\(P\\).

### Уравнение Ферхюльста (Логистический рост)
*   Классическая модель ограниченного роста (1838 г.).
*   **Уравнение:** \\(\frac{dx}{dt} = rx(1 - \frac{x}{K})\\).
*   **Свойства:**
    1.  При малых \\(x\\) рост близок к экспоненциальному.
    2.  При больших \\(x\\) численность приближается к пределу \\(K\\) (емкость среды).
*   **Стационарные точки:** \\(\bar{x}_1 = 0\\) (неустойчива) и \\(\bar{x}_2 = K\\) (устойчива).

## 7. Решение линейных уравнений 1-го порядка
*   **Общий вид:** \\(\frac{dx}{dt} + P(t)x + Q(t) = 0\\).
*   **Однородное (\\(Q=0\\)):** Решается разделением переменных. Общий вид: \\(x(t) = C \exp(-\int P(t)dt)\\).
*   **Неоднородное (\\(Q \neq 0\\)):** Решается методом вариации произвольной постоянной.

## 8. Важные выводы лекции
1.  **Ограниченность:** Одним автономным дифференциальным уравнением можно описать только **монотонные** изменения переменной.
2.  **Невозможность:** Такие уравнения **не могут** описывать периодические (колебания) или хаотические процессы.
3.  Для моделирования более сложного поведения (колебаний, хаоса) необходимо:
    *   Переходить к системам высших порядков (2-го и выше).
    *   Вводить время в явном виде.
    *   Использовать дискретные уравнения или уравнения с запаздыванием.
