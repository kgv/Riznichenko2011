# Лекция 4: Модели, описываемые системами двух автономных дифференциальных уравнений

### 1. Введение и основные понятия
Основы качественного исследования систем на фазовой плоскости заложены **Анри Пуанкаре**. Этот метод позволяет изучать свойства биологических систем без поиска точного аналитического решения уравнений.

**Общий вид системы (автономная система):**
$$\frac{dx}{dt} = P(x, y)$$
$$\frac{dy}{dt} = Q(x, y)$$
Где $P$ и $Q$ — непрерывные функции, имеющие непрерывные производные.

*   **Фазовая плоскость:** плоскость с осями координат $x$ и $y$, где каждой точке соответствует определенное состояние системы.
*   **Изображающая (представляющая) точка:** точка $M(x, y)$, отображающая состояние системы в данный момент времени.
*   **Фазовая траектория:** путь, который описывает изображающая точка при изменении состояния системы во времени.
*   **Фазовый портрет:** совокупность фазовых траекторий при различных начальных условиях. Дает наглядное представление о поведении системы.

---

### 2. Метод изоклин
Используется для построения фазового портрета.
*   **Изоклина:** линия на фазовой плоскости, во всех точках которой касательные к фазовым траекториям имеют одинаковый наклон ($dy/dx = A = \text{const}$).
*   **Главные изоклины:**
    1.  $Q(x, y) = 0$ — изоклина горизонтальных касательных ($dy/dx = 0$).
    2.  $P(x, y) = 0$ — изоклина вертикальных касательных ($dy/dx = \infty$).
*   **Особая точка (точка покоя):** точка пересечения главных изоклин, где $dx/dt = 0$ и $dy/dt = 0$. Она соответствует **стационарному состоянию** системы.

---

### 3. Устойчивость стационарного состояния
*   **Определение по Ляпунову:** состояние равновесия устойчиво, если при малом отклонении от него система со временем не уходит далеко от этой точки (находится в пределах заданной области $\varepsilon$).
*   **Грубые системы:** системы, характер поведения которых не меняется при малых изменениях их параметров.

---

### 4. Классификация особых точек (линейные системы)
Тип стационарного состояния определяется корнями ($\lambda_1, \lambda_2$) характеристического уравнения:
$$\lambda^2 - (a + d)\lambda + (ad - bc) = 0$$

| Тип особой точки | Характер корней ($\lambda$)                          | Поведение системы                                                                                                 |
| :--------------- | :--------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------- |
| **Узел**         | Действительные, одного знака                         | Траектории входят в точку (устойчивый, если $\lambda < 0$) или выходят из нее (неустойчивый, если $\lambda > 0$). |
| **Седло**        | Действительные, разных знаков                        | Всегда **неустойчиво**. Траектории приближаются к точке по одной оси и удаляются по другой.                       |
| **Фокус**        | Комплексно-сопряженные ($\text{Re } \lambda \neq 0$) | Траектории имеют вид спиралей. Соответствует затухающим (устойчивый) или нарастающим (неустойчивый) колебаниям.   |
| **Центр**        | Чисто мнимые ($\text{Re } \lambda = 0$)              | **Негрубое** состояние. Траектории — замкнутые эллипсы. Колебания постоянной амплитуды.                           |

---

### 5. Бифуркационная диаграмма
Для анализа используется плоскость параметров $\sigma$ и $\Delta$:
*   $\sigma = -(a + d)$
*   $\Delta = ad - bc$ (детерминант)

**Условие устойчивости:** $\sigma > 0$ и $\Delta > 0$ (первая четверть диаграммы).
**Бифуркация:** качественное изменение фазового портрета при изменении параметров системы.
*   **Бифуркация Андронова–Хопфа:** переход от устойчивого фокуса к неустойчивому через центр, что в нелинейных системах приводит к рождению **предельного цикла** (автоколебаний).

---

### 6. Практический пример: Линейные химические реакции
Рассматривается схема: приток $X \to$ превращение в $Y \to$ отток $Y$.
*   **Система уравнений:**
    $dx/dt = k_1 - k_2x$
    $dy/dt = k_2x - k_3y$
*   **Анализ:**
    1.  Стационарные значения: $\bar{x} = k_1/k_2, \bar{y} = k_1/k_3$.
    2.  Корни характеристического уравнения: $\lambda_1 = -k_2, \lambda_2 = -k_3$.
*   **Результат:** так как оба корня действительны и отрицательны, особая точка — **устойчивый узел**. Колебания в такой системе невозможны.
